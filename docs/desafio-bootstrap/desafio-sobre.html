Sobre o Desafio

O objetivo desta atividade é tornar o aluno protagonista do seu próprio aprendizado sobre o Bootstrap. O aluno deverá construir esta página utilizando como base as ferramentas oferecidas para construir páginas responsivas, com componentes previamente prontos para facilitar a vida dos desenvolvedores.

Com esta atividade o aluno irá pesquisar no site do Bootstrap e encontrar os elementos utilizados nesta página, devendo testá-los e configurá-los para construir um site com essas mesmas características.

Ao final espera-se que os alunos se ambientem na documentação da ferramenta, tendo mais liberdade para explorar e descobrir todo o potêncial oferecido além de praticar os conceitos já aprendidos sobre HTML5 e CSS3.

Este é o rodapé da atividade bootstrap
